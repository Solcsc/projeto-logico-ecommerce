# projeto-logico-ecommerce
Refinamento de um projeto conceitual de banco de dados para E-commerce
Cliente PF e PJ : Diferenciação entre Pessoa Física e Jurídica com documentos específicos (CPF/CNPJ)
Gestão de Pagamento : Possibilidade de um cliente ter múltiplas formas de pagamento cadastradas.
Entrega : Adição de informações de entrega vinculadas ao pedido, com status e códido de rastreio.
